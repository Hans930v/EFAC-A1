# Schematics

This folder contains the **circuit schematics and wiring diagrams** for the EFAC‑A1 project.  
These design sources document the electrical connections, sensor integration, and control logic used in the External Feeder–Assisted Filament Change system.

## Purpose
- Provide clear reference for hardware design and integration.  
- Ensure reproducibility of EFAC‑A1 builds.  
- Support community modifications and improvements.
