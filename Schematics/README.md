# Schematics

This folder will contain the **circuit schematics and wiring diagrams** for the EFAC‑A1 project.  
These design sources will document the electrical connections, sensor integration, and control logic used in the External Feeder–Assisted Filament Change system.

## Purpose
- It will provide clear reference for hardware design and integration.  
- It will ensure reproducibility of EFAC‑A1 builds.  
- It will support community modifications and improvements.
