# Electronics

This folder contains the **electronic design files and documentation** for the EFAC‑A1 project.  
It includes the Bill of Materials (BOM), schematics, wiring diagrams, and PCB layouts that define the electrical side of the External Feeder–Assisted Filament Change system.

## Purpose
- Provide a clear reference for all required electronic components.  
- Document schematics and wiring for reproducibility.  
- Support expansion modules and community modifications.  

## Structure
- **BOM.md** → Full Bill of Materials with sourcing links and notes.  
- **schematics/** → Circuit diagrams and wiring references.  
- **pcb/** → PCB layouts and design files (if applicable).  

## Bill of Materials
The BOM is split into two sections:
- **Main System (Base, 4 Colors)** → Required electronics for the core EFAC‑A1 unit.  
- **Stacking Unit (Expansion)** → Additional electronics for extending filament capacity.  

See [BOM.md](BOM.md) in this folder for the complete list with sourcing links and images.
