# G‑code

This folder contains the **Change Filament G‑code** files for the EFAC‑A1 project.

> ⚠️ **Disclaimer**  
> This is **NOT** official Bambu Lab G‑code.  
> These are community‑developed, experimental files intended for use with EFAC‑A1 hardware.

## File
- [**EFAC‑A1 Change Filament G‑code**](EFAC-A1-Gcode.gcode)  
  Enables AMS‑like filament changes via the external feeder, using EFAC‑A1 hardware and the PESA protocol.
